# code to select sort
### 选择排序/冒泡排序：选择排序是在一组数列当中，选择一个数，然后用数列的其它值和它依次比较；这个过程持续多次，直到遍历全部数组就完成了排序。
### 插入排序：以升序为例，插入排序是选一个数和前面的数进行比较，如果这个数比前面的数小，那么前面的数进行后移，直到前面的数比它小，那么这个数就插入到比它小数据的后一位。
### 希尔排序：希尔排序是插入排序的优化，这个排序方法运用了步长，通过跳着比较数据大小来减少操作步数，与插入排序相比，希尔排序减少了数据位移的步骤，减少了程序的运行复杂度。
### 快速排序：快速排序是一个递归的排序，提出了游标和基数的概念，通过游标指到的数来与基数进行比较，通过一个递归的过程来实现数据的排序。
### 基数排序：基数排序又称桶排序，根据数据的个位、百位、千位放入数组当中，再根据特定的顺序从数组当中读出数据来实现数据的排序。
